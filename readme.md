# Beyond Positive or Negative: How Mixed Emotions on Reddit Predict Depressive Symptoms in Asian American Families

This repository contains the code and data resources for the project **“Beyond Positive or Negative: How Mixed Emotions on Reddit Predict Depressive Symptoms in Asian American Families.”** The project investigates whether mixed emotional expressions in Reddit posts can help predict depressive symptoms in the context of Asian American family experiences.
